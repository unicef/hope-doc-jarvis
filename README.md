# HOPE Jarvis

RAG Bot for UNICEF HOPE documentation.

## Setup

```bash
pip install -e .
```

## Usage

```bash
# Sync all repos
jarvis sync

# Update single repo
jarvis update <repo_name>

# Start webhook server
python -m hope_jarvis.webhook
```
